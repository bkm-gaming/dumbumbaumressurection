Minecraft Java Edition - Schritt für Schritt

1

Minecraft Java Edition öffnen

Starte Minecraft Java Edition auf deinem PC. Stelle sicher, dass du die Version 1.9 oder neuer installiert hast, da unsere Server mit diesen Versionen kompatibel sind.

Tipp: Du kannst die Version im Minecraft Launcher unter "Installations" auswählen oder eine neue Installation erstellen.

2

Zum Multiplayer-Menü navigieren

Klicke im Hauptmenü auf "Multiplayer" und dann auf "Server hinzufügen". Hier kannst du die Serverdetails eingeben.

3

Serverinformationen eingeben

Gib einen Namen für den Server ein (z.B. "DBBR") und trage die Serveradresse ein. Klicke auf den Server oben, um die IP automatisch zu übernehmen: -----:)

4

Server beitreten

Klicke auf "Fertig" und dann auf den neu erstellten Server in deiner Liste. Drücke "Server beitreten" und du wirst mit unserer Welt verbunden.

5

Erste Schritte auf dem Server

Nach dem Beitreten erscheinst du am Spawnpunkt. Folge den Hinweisschildern, um die Regeln zu lesen und das Starter-Kit zu erhalten. Viel Spaß beim Spielen!

Minecraft Bedrock Edition - Schritt für Schritt

1

Minecraft Bedrock öffnen

Starte Minecraft auf deinem Gerät (Windows 10/11, Xbox, PlayStation, Switch, Mobile). Stelle sicher, dass du die neueste Version installiert hast.

2

Zum Servers Tab navigieren

Gehe zum "Play" Menü und wähle dann den Tab "Servers" aus. Scrolle nach unten zu "Server hinzufügen".

3

Serverinformationen eingeben

Gib einen Servernamen ein (z.B. "DumBumBaum 1 Reborn") und trage die Serveradresse ein. Klicke auf den Server oben, um die IP automatisch zu übernehmen: DumBumBaum.aternos.me

Port: 20196 (für DumBumBaum 1 Reborn)

4

Server speichern und beitreten

Klicke auf "Speichern" und dann auf den Server in deiner Liste. Wähle "Server beitreten" aus, um die Verbindung herzustellen.

5

Erste Schritte auf dem Server

Nach erfolgreicher Verbindung erscheinst du am Bedrock-Spawnpunkt. Hier findest du alle wichtigen Informationen und dein Starter-Kit.
